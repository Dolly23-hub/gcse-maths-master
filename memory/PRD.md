# GCSE Maths Master - PRD (Final v2)

## Problem Statement
Comprehensive GCSE Maths revision app for UK students covering all exam boards (Edexcel, AQA, OCR) with topic explanations, past papers (2020-2024), quizzes, formulas, AI tutoring, and personalised revision planner.

## Architecture
- **Frontend**: React + Tailwind CSS + Shadcn UI
- **Backend**: FastAPI + MongoDB
- **AI**: OpenAI GPT-5.2 via Emergent LLM Key (explanations + revision tips)

## Content Summary
- **33 topics** across 5 categories (incl. Higher: Surds, Circle Theorems, Vectors, etc.)
- **84 quiz questions** with instant feedback
- **38 past papers** covering 2020-2024 (Edexcel: 15, AQA: 12, OCR: 11)
- **30 formulas** with usage examples

## Features
1. Topic Explanations (markdown-rendered, worked examples, key points)
2. Past Papers Hub (tabbed Edexcel/AQA/OCR, expandable practice Qs, 5 year range)
3. Interactive Quiz System (topic-specific or mixed, scoring, explanations)
4. Formula Sheet (searchable, expandable, grouped by category)
5. AI Maths Tutor (GPT-5.2 chat with suggested questions)
6. Revision Planner (exam board/date/confidence input, AI tips, weekly schedule)
7. Exam Countdown (2026 dates with days remaining)
8. Progress Tracking (localStorage-based topic viewing + quiz scores)
9. Mobile Responsive navigation
